# Design Requirements

- Keep all badges the same size.
- Arrange them in a clean responsive grid.
- Desktop: 3 badges per row.
- Tablet: 3 badges per row.
- Mobile: 2 badges per row.
- Use generous spacing between badges.
- Preserve the original aspect ratio.
- Do not crop or stretch the images.
- Use a subtle hover effect (slight scale and soft glow).
- The section should remain minimal, clean, and professional.
