# Getting Started

Clone the repository

```bash
git clone https://github.com/rukeshsg/rukesh-portfolio.git
```

Move into the project

```bash
cd rukesh-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
